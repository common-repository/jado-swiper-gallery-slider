Plugin creates automatically Swiper-Slider for WordPress galleries.

== Description ==
Plugin creates automatically Swiper-Gallery-Slider for WordPress galleries. Swiper is a great Slider from idangero.us.

This plugin just transforms automatically the WordPress gallery to an Swiper-Slider if plugin is activated. Very simple & easy to use. 

Swiper - is the free and most modern mobile touch slider with hardware accelerated transitions. It is built to be used in desktop & mobile websites.



= Features include =

* Fully responsive
* Touch-enabled navigation
* Keyboard control
* Slider Vertical
* Fade, cube or coverflow effect
* Drag & Drop order change through the WordPress gallery
* Responsive height control
* Image resolution
* With or without thumbnails
* Pagination count
* Autoplay
* Hash navigation



== Installation ==

1. Upload the 'jado-swiper-gallery-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the "jado-swiper-gallery-slider" plugin through the 'Plugins' menu in WordPress.
